Key behaviours in this view

  • Inline price edit — click the amber number, type new value, press Enter or blur to auto-save via PATCH /api/menu/[id]
  • Inline stock edit — same pattern. Colour: green ≥ 6, amber 1–5 (low), red = 0 (auto-marks unavailable)
  • Stock = 0 auto-disables — when stock hits 0, availability toggle flips to OFF automatically. Staff must restock to re-enable
  • Low-stock strip — pinned at top, lists every item below the threshold (default: 5). Clicking "Jump to items" filters the table to show only low-stock rows
  • Availability toggle — instant save, no drawer needed. Disabled (greyed out) when stock = 0
  • Import CSV — the existing MenuImport component moves here as a button. Opens a modal, unchanged functionality
  • Stock auto-decrement — when a POS order is confirmed (createOrder), each ordered item's stock decrements by quantity. Handled in pos.actions.ts

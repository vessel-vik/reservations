Finance Hub — Which layout approach?

Three ways to structure the unified Accounting + VAT + Expenses hub

A

Inner-Tab Finance Hub

Classic sub-navigation inside the Finance tab. Overview | Expenses | VAT | Reports each fill the full view. Budget alerts appear as a banner inside the Expenses section.

B

Overview-First Finance Hub ⭐ Recommended

A persistent KPI strip stays pinned at the top in all sections. Below it, a segmented control switches between Expenses / VAT / Reports. The admin always sees P&L while working.

C

Command Centre (All-in-One)

No sub-navigation at all. Overview KPIs + trend chart on the left column. Expense entry + list on the right. VAT summary and budget meters inline below. One scroll, everything visible.

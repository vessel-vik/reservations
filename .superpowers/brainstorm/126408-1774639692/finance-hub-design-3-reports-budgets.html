New infrastructure this section introduces

  • Budgets Appwrite collection — fields: category, monthlyLimit, month, year. One document per category per month.
  • /api/budgets route — GET (fetch month's budgets), POST/PUT (set/update limits)
  • Recharts (already standard in Next.js stacks) for the bar + line chart — or a lightweight SVG chart if we want zero dependencies
  • CSV + PDF export — CSV is a simple client-side string build; PDF via browser window.print() with a print stylesheet
  • Auto-carry-forward — when opening Set Budgets for a new month with no budgets set, it pre-fills from the previous month

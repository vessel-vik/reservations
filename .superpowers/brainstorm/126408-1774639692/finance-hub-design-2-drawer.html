Current (broken)

  • Edit button imported but never rendered
  • Amount defaults to 0, passes validation
  • No receipt attachment
  • No VAT preview while typing
  • Mismatched styles (gray-800 vs slate-800)
Rebuilt drawer
  • Zod validation — inline field errors
  • Button shows spinner + disables during submit
  • Toast on success ("Expense saved") or error

  • Amount must be > 0 with clear error
  • Live VAT breakdown as you type the amount
  • Receipt upload via Appwrite Storage

Cross-table order combining — how should it work?

When a customer wants to pay for friends at another table, how does the waiter find those orders?

A

Pull by table number

Waiter types "Table 5" → that table's unpaid orders get added to the current bill. Fast when the waiter knows exactly where the friends are sitting.

B

Browse all open orders

A panel shows every unpaid order across all tables. Waiter taps any order to include it. Best for bar seating or when tables aren't numbered clearly.

C

Both — table lookup + open orders browser

Start from a table as usual, then tap "Add from another table" to either type a table number OR browse all open orders. Most flexible.
